# localai-onnx

Minimal LocalAI v4 backend scaffold for running `onnx-asr` speech-to-text models.

Included in this repo:

- `backend/python/onnx-asr/` - Python gRPC backend
- `Dockerfile` - OCI image build for the backend
- `gallery/index.yaml` - LocalAI backend gallery definition

Start with `backend/python/onnx-asr/README.md`.
